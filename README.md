## General info
When you enter website it detects your ip address and shows your approximate location

## Technologies
Project is created with:
### Front-End
* parcel-bundler 1.12.5
* bootstrap-icons 1.8.0
* SCSS/SASS


## Setup
To run this project:
* ```$npm i -D parcel-bundler```
* ```$npm run dev```

To create a build:
* ```$npm run build```
